# Code Complexity Analyzer

Xcode 프로젝트의 코드 복잡도를 분석하는 macOS 앱입니다.

## 주요 기능

### 📊 분석 지표
- **라인 수**: 파일별 코드 라인 수 (공백 제외)
- **함수 개수**: func, init 포함
- **타입 개수**: Class, Struct, Enum, Protocol 개수
- **프로퍼티 개수**: var, let 선언 개수
- **순환 복잡도**: if, guard, for, while, case 등의 분기문 개수
- **복잡도 점수**: 위 지표들을 가중치 적용하여 계산한 종합 점수

### 🎯 복잡도 레벨
- **낮음** (0-50점): 유지보수 쉬움
- **보통** (50-150점): 적정 수준
- **높음** (150-300점): 리팩토링 검토 필요
- **매우 높음** (300점 이상): 즉시 리팩토링 필요

### 🔍 기능
1. **프로젝트 분석**: Xcode 프로젝트 폴더 선택 시 자동으로 모든 Swift 파일 분석
2. **정렬 옵션**: 복잡도, 라인 수, 함수 수, 이름순 정렬
3. **검색**: 파일 이름으로 빠른 검색
4. **상세 정보**: 각 파일 클릭 시 상세 분석 정보 표시
5. **시각화**: 복잡도 게이지, 색상 코딩으로 직관적 표시
6. **요약 정보**: 프로젝트 전체 통계 표시

## 사용 방법

### Xcode에서 빌드 및 실행

1. `CodeComplexityAnalyzer.xcodeproj` 파일을 Xcode로 열기
2. 타겟을 "My Mac"으로 선택
3. ⌘+R로 실행

### 앱 사용

1. 앱 실행 시 자동으로 폴더 선택 다이얼로그 표시
2. 분석할 Xcode 프로젝트 폴더 선택
3. 분석 완료 후 결과 확인
4. 각 파일을 클릭하여 상세 정보 확인
5. 정렬/검색으로 원하는 파일 찾기

## 분석 제외 항목

다음 디렉토리는 자동으로 분석에서 제외됩니다:
- `.build/`
- `Pods/`
- `DerivedData/`
- `.swiftpm/`

## 복잡도 점수 계산 방식

```
복잡도 점수 = (라인 수 × 0.1) 
           + (함수 개수 × 2.0)
           + (타입 개수 × 3.0)
           + (순환 복잡도 × 1.5)
```

## 활용 방법

### 리팩토링 우선순위 결정
- 복잡도 점수가 높은 파일부터 리팩토링
- "매우 높음" 파일은 즉시 분할 검토

### 코드 리뷰 효율화
- 복잡한 파일에 더 많은 리뷰 시간 할당
- 신규 기능 추가 시 복잡도 증가 모니터링

### 기술 부채 관리
- 정기적으로 분석하여 복잡도 추이 확인
- 목표 복잡도 수준 설정 및 관리

### 팀 코드 품질 관리
- 복잡도 임계값 설정
- PR 전 복잡도 체크

- Xcode 15.0 이상 (빌드용)
